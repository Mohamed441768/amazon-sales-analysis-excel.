# Amazon Sales Analysis (Excel Project)

This project is a data analysis case study of **Amazon sales** using **Microsoft Excel**.  
It includes cleaning, exploration, pivot tables, and dashboards to uncover useful business insights.

## 📊 Project Overview

The dataset contains sales records from Amazon, and the analysis focuses on:
- Identifying the top-selling products
- Exploring sales by city
- Understanding customer payment preferences
- Creating a clear and informative dashboard

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations

## 📌 Key Insights
- Best-selling products across regions
- Cities with the highest revenue
- Popular payment methods
- Clean, visual summary using dashboards

- `Data analysis for Amazone Excel.xlsx` — The main analysis workbook
- `top_selling_products.png` — Bar chart of top-selling products
